### 문제 설명

&lt;p>좌표평면을 좋아하는 진수는 x축과 y축이 직교하는 2차원 좌표평면에 점을 찍으면서 놀고 있습니다. 진수는 두 양의 정수 &lt;code>k&lt;/code>, &lt;code>d&lt;/code>가 주어질 때 다음과 같이 점을 찍으려 합니다.&lt;/p>

&lt;ul>
&lt;li>원점(0, 0)으로부터 x축 방향으로 &lt;code>a*k&lt;/code>(a = 0, 1, 2, 3 ...), y축 방향으로 &lt;code>b*k&lt;/code>(b = 0, 1, 2, 3 ...)만큼 떨어진 위치에 점을 찍습니다.&lt;/li>
&lt;li>원점과 거리가 &lt;code>d&lt;/code>를 넘는 위치에는 점을 찍지 않습니다.&lt;/li>
&lt;/ul>

&lt;p>예를 들어, &lt;code>k&lt;/code>가 2, &lt;code>d&lt;/code>가 4인 경우에는 (0, 0), (0, 2), (0, 4), (2, 0), (2, 2), (4, 0) 위치에 점을 찍어 총 6개의 점을 찍습니다.&lt;/p>

&lt;p>정수 &lt;code>k&lt;/code>와 원점과의 거리를 나타내는 정수 &lt;code>d&lt;/code>가 주어졌을 때, 점이 총 몇 개 찍히는지 return 하는 solution 함수를 완성하세요.&lt;/p>

### 풀이

```java
class Solution {
    
    public long solution(int k, int d) {
        
        long answer = 0;
        
        for (long x=0; x<=d; x+=k) {
            long maxY = (long) Math.sqrt((long)d * d - (long)x * x);
            answer += maxY/k+1;
        }

        return answer;
    
    }
    
}
```
